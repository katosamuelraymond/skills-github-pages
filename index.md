# MediConnect Africa 🌍

**MediConnect Africa** is a high-end, accessible community health navigator designed for low-data environments across Africa.

## 🚀 Live Demo
[View the Live Application](https://app-b0oqomon7rwh.appmedo.com)

## 🛠 Tech Stack
- **Frontend:** React, Next.js, Tailwind CSS, Shadcn UI
- **Logic:** State-Driven Decision Trees & Session Memory
- **Accessibility:** Web Speech API (Voice-to-Text)
- **Design:** Professional Glassmorphism

## ✨ Core Features
- **Multilingual Support:** Instant switching between Swahili, Hausa, and English.
- **AI Symptom Triage:** Real-time urgency ratings and guidance.
- **Clinic Locator:** Dynamic routing to the nearest healthcare facilities.
- **Audio Feedback:** Real-time text-to-speech for low-literacy accessibility.
